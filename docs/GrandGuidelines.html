🎯 Brand Overview

Usage Guidelines

✓ Always Do

  • Use official logo files (never recreate)
  • Maintain 15% clear space around logo
  • Ensure 4.5:1 contrast ratio minimum
  • Use correct color version for background
  • Keep aspect ratio locked

✗ Never Do

  • Don't add effects (shadows, glows, outlines)
  • Don't change colors outside palette
  • Don't rotate, skew, or distort
  • Don't place on busy backgrounds
  • Don't use below minimum sizes
⚠️ Minimum Sizes
  • Full logo: 120px wide (ensures handshake detail visible)
  • Icon only: 32px × 32px minimum
  • Favicon: 16px (may need simplified version)

CSS Variables for Developers

Usage Guidelines

  • 80/20 Rule: 80% warm tones (browns/creams), 20% black for high contrast text
  • WCAG Compliance: Always ensure 4.5:1 contrast for text, 3:1 for UI elements
  • Primary CTA: Rich Brown (#584840) background + Cream text
  • Backgrounds: Cream for main pages, Light Beige for sections/cards
  • Never: Bright reds, blues, or greens—maintain warm palette integrity

### 7. FAQ Section (IMPROVED)
**Background:** Light Beige (#F8F5ED)

**NEW Questions Based on Competitor Research:**

#### Q1: How is Quoffer different from other RFQ apps?
```
A: Quoffer is built on Shopify's native app embed system, which means 
it won't break your theme and can be enabled/disabled with one click. 
We focus on simplicity—our guided setup gets you live in under 10 minutes. 
Plus, we'll never charge commission fees on your sales.

Unlike apps that inject code into your theme, Quoffer uses Shopify's 
official extension system for guaranteed compatibility and easy removal.
```

#### Q2: Will this slow down my store or break my theme?
```
A: No. Quoffer uses Shopify's native app embed system (the same technology 
used by Shopify's own features). This means:

✓ Zero impact on page load speed
✓ No theme code modifications
✓ 100% compatible with any theme
✓ Works with theme updates
✓ Uninstalls cleanly (no leftover code)

Many older RFQ apps use "Liquid injection" which can break themes. 
Quoffer is modern and theme-safe by design.
```

#### Q3: Can I hide prices for some customers but not others?
```
A: Yes! This is one of Quoffer's core features. You can control price 
visibility based on:

• Customer tags (e.g., "Wholesale", "VIP", "B2B")
• Login status (show to logged-in, hide from guests)
• Product/collection (hide specific products only)
• Any combination of the above

Example: Show retail prices to regular shoppers, hide prices for 
tagged wholesale customers, and show "Request Quote" button instead.
```

#### Q4: Do customers need to log in to request a quote?
```
A: No, but you can require it if you want. By default, both guests and 
logged-in customers can request quotes. If you enable "Login required," 
only logged-in customers will see the quote button.

Benefits of requiring login:
• Auto-fill customer data (saves them time)
• Track quote history per customer
• Better for B2B customer management
```

#### Q5: How do quotes become actual orders?
```
A: Simple three-click process:

1. You review the quote in your dashboard
2. Click "Create Draft Order" (pre-filled with customer + product details)
3. Customer receives Shopify checkout link via email

The Draft Order appears in your Shopify admin just like any other order. 
When the customer pays, it converts to a regular order and enters your 
fulfillment workflow. No manual invoice creation, no discount codes to 
manage.
```

#### Q6: Can I customize the form fields?
```
A: Yes, unlimited customization. Add any fields you need:

• Text inputs (Tax ID, Company Name, etc.)
• Text areas (Project details, Special requirements)
• Dropdowns (Delivery options, Product variations)
• Checkboxes (Terms acceptance, Feature selections)
• File uploads (Blueprints, Design files, Specifications)

Each field can be:
• Required or optional
• Hidden based on conditions
• Pre-filled for logged-in customers
• Styled to match your brand
```

#### Q7: Is there a contract or monthly commitment?
```
A: No contracts. No commitments. 

• Start with our free plan (no credit card required)
• Upgrade when you're ready
• Downgrade anytime
• Cancel anytime

We believe in earning your business through a great product, 
not locking you in with contracts.
```

#### Q8: What kind of support do you offer?

A: All plans include email support with average response time under

We also provide:

## CONTENT WRITING GUIDELINES (IMPROVED)

### Voice Principles

**DO:**
✅ Lead with benefits, not features ("Turn quotes into sales" not "Quote management system")
✅ Use concrete examples ("Hide prices for wholesale customers" not "Price visibility control")
✅ Address pain points directly ("No theme code modification" not "Uses app embed")
✅ Use numbers and specifics ("5 minutes" not "quickly", "$5/mo" not "affordable")
✅ Write in active voice ("You review quotes" not "Quotes are reviewed")
✅ Use "you" language (focus on merchant, not us)

**DON'T:**
❌ Use jargon without explanation
❌ Make vague claims ("best", "revolutionary", "amazing")
❌ Write long paragraphs (keep to 2-3 sentences max)
❌ Assume technical knowledge
❌ Use corporate-speak or formal language

### Benefit-Focused Headlines
```
Instead of: "Quote Request Form"
Write: "Collect Exactly What Your Business Needs"

Instead of: "Price Hiding Feature"
Write: "Protect Wholesale Margins and Encourage Inquiries"

Instead of: "Dashboard Management"
Write: "Respond Faster, Close More Deals"
```

### Use Case Examples
Always include relatable examples:
```
Generic: "Perfect for B2B sellers"
Better: "I sell industrial equipment. Wholesale buyers need custom 
pricing based on quantity and delivery location."
```
